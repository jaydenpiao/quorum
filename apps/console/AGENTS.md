# AGENTS.md — operator console area

## Goal

Keep the console simple and operational.

The console should show:
- intents
- findings
- proposals
- votes
- policy status
- execution status
- rollback status
- event timeline

## Rules

- do not turn the console into a chat app
- prefer tables, cards, and timelines
- keep it readable from raw HTML/JS
- align visible states to backend state names
- if backend state changes, update labels here
